隐私政策

最后更新：2025年1月12日

感谢您使用本应用！我们非常重视您的隐私权益。

核心原则：本应用不收集、不存储、不传输任何个人数据到服务器。所有数据均保存在您的设备本地。

1. 数据收集

本应用不收集任何个人数据。

我们不会：

  • 收集您的个人身份信息
  • 收集您的屏幕使用时间数据
  • 收集您的浏览历史
  • 收集您的位置历史记录
  • 使用任何第三方分析或追踪工具
  • 向任何第三方共享您的信息

2. 数据存储

以下数据仅存储在您的设备本地，不会上传到任何服务器：

  • 应用分组配置：您创建的应用分组和选择的应用列表
  • 时间日程设置：您配置的定时限制/允许使用规则
  • 地理围栏位置：您设置的特定地点坐标和半径（如果使用此功能）
  • 应用设置：您的偏好设置和应用锁定密码（如果设置）

3. 权限使用说明

3.1 屏幕使用时间权限

用途：用于在专注时段临时限制您选择的应用使用，帮助您保持专注。

  • 仅使用应用选择和显示控制功能
  • 不访问屏幕使用时间统计数据
  • 不访问应用使用历史
  • 所有操作完全由您控制

3.2 位置服务权限（可选）

用途：用于基于地理位置的自动应用管理功能。

  • 此功能完全可选，您可以选择不使用
  • 位置数据仅用于判断是否进入/离开设定区域
  • 位置信息仅存储在本地
  • 不记录或上传位置历史

3.3 Face ID / Touch ID（可选）

用途：用于应用锁定功能，保护隐私设置。

  • 此功能完全可选
  • 生物识别数据由iOS系统处理，本应用无法访问
  • 仅用于验证身份，不存储任何生物识别信息

4. 数据安全

由于所有数据都存储在您的设备本地：

  • 数据安全由iOS系统的沙盒机制保护
  • 如果您设置了设备密码或Face ID，数据会被加密
  • 应用卸载后，所有本地数据将被自动删除
  • 我们无法访问或恢复您的数据

5. 第三方服务

本应用不使用任何第三方服务，包括：

  • 不使用第三方分析工具
  • 不使用第三方广告服务
  • 不使用第三方崩溃报告服务
  • 不集成任何社交媒体SDK

6. 您的权利

由于我们不收集任何数据，您完全控制设备上的所有本地数据，可以随时：

  • 在应用内查看和修改所有设置
  • 通过卸载应用删除所有本地数据
  • 在系统设置中撤销任何权限

7. 联系我们

如果您对本隐私政策有任何疑问，请通过应用内的"意见反馈"功能联系我们。
